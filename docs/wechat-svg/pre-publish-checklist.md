# 发布前自检清单

交付或发布前对照这份清单逐项检查。这是防止"本地看着好,发出来就坏"的最后一道闸。

## 一、技术兼容性

### SVG 动画层
- [ ] 所有 `<animate>` / `<set>` / `<animateTransform>` 的 `attributeName` 都在白名单 20 项内
- [ ] 没有 `repeatCount="indefinite"` 作用在 `height` / `width` 上
- [ ] 需要停留终态的动画都写了 `fill="freeze"`
- [ ] `begin="elementId.click"` 引用的 id 在文章内唯一
- [ ] `animateTransform` 的 `type` 属性值正确(translate/scale/rotate/skewX/skewY)

### HTML/CSS 层
- [ ] 没有使用 `position: absolute` / `fixed`
- [ ] 没有使用 `clip-path`、`mask`、`filter`(CSS 层)、`backdrop-filter`、`mix-blend-mode`
- [ ] 没有使用 `<script>` 或 `onclick` 等事件处理器
- [ ] 没有使用 `<iframe>`、`<embed>`、`<object>`
- [ ] 没有使用 CSS 变量(`var(--xxx)`),颜色都是写死的值
- [ ] `margin` 负值没有超过容器宽度
- [ ] 没有使用 CSS Grid

### 命名空间和声明
- [ ] 每个 `<svg>` 都声明了 `xmlns="http://www.w3.org/2000/svg"`
- [ ] 使用 `<foreignObject>` 时,内部 `<div>` 声明了 `xmlns="http://www.w3.org/1999/xhtml"`
- [ ] `<svg>` 标签有 `style="display:block;"` 避免行高间隙

### 响应式
- [ ] SVG 容器 `width: 100%`
- [ ] `viewBox` 设置正确,在不同设备宽度下不变形
- [ ] `preserveAspectRatio` 根据内容需要设置(默认 `xMidYMid meet`)

## 二、交互可用性

- [ ] 每个交互都有视觉提示(按钮样式、hover 效果、文字提示)
- [ ] 触发区域足够大(移动端最小 44x44 像素的可点击区域)
- [ ] 交互完成后有反馈(动画、文字变化、跳转)
- [ ] 没有"死链"——所有 `<a href>` 都指向有效 URL
- [ ] 交互不会阻断阅读流——用户不点击也能看完文章

## 三、内容质量

### 文案
- [ ] 标题具体,有数字或悬念,不用"干货"/"必看"
- [ ] 开篇前 100 字回答了"这跟我有什么关系"
- [ ] 段落都在 4 行以内
- [ ] 感叹号不超过 3 个
- [ ] 没有"高效便捷智能专业"这种空话
- [ ] 中英文之间有空格
- [ ] 使用中文标点符号(「」""—— ……)
- [ ] 读出声来语气自然

### 视觉
- [ ] 图文节奏合理,每 200-300 字至少一个视觉元素
- [ ] SVG 交互点 1-3 个,不过载
- [ ] 品牌色使用一致
- [ ] 字体在目标平台有兜底(PingFang SC, Microsoft YaHei 等)

## 四、品牌和合规

- [ ] 品牌 logo、名称、颜色符合品牌规范
- [ ] 版权声明(如需要)
- [ ] 涉及产品/服务的宣称有数据支撑,不做绝对化承诺("最好"、"第一"等词慎用)
- [ ] 涉及引用的内容有来源说明
- [ ] 涉及个人信息/数据的内容符合相关法规
- [ ] 没有未经授权使用的图片/字体/字库

## 五、发布前的真实环境测试

### 必做
- [ ] 在浏览器(Chrome/Safari)预览过完整效果
- [ ] 在微信编辑器里**真实预览**过(不是"保存草稿",是"发给自己预览")
- [ ] iOS 和 Android 各测过一次
- [ ] 点过所有 CTA 按钮,跳转正确
- [ ] 查过文章在订阅号列表里的标题+摘要显示效果

### 强烈建议
- [ ] 找一个不了解产品的朋友读一遍,问 TA:"读完知道这篇在说什么、要你做什么吗?"
- [ ] 深色模式下看一眼(即便效果会劣化,至少确认不会完全崩掉)
- [ ] 检查不同机型(老 Android、新 iPhone)的表现

## 六、发布后 24 小时监测

虽然不是"发布前",但值得规划:

- [ ] 发布后 1 小时内检查阅读量异常是否意味着封面/标题有问题
- [ ] 关注留言区是否有"动画不显示"等反馈
- [ ] 记录完读率、CTA 点击率,为下一篇迭代

## 一句话总结

**"在微信编辑器里预览过、iOS 和 Android 都看过、有一个朋友读过"——这三件事做完再点发送**。
